Mountain hut availability map

This is a map of mountain huts that can be booked via hut-reservation.org. It shows indicative availability based on the latest automatic data update.

This is a small private project. The data may be incomplete, contain errors, or become outdated. The official reservation system of each hut is always the source of truth.

If you find a mistake or something does not work, please let me know or open an issue:
github.com/ekowork/alpine-huts-map/issues

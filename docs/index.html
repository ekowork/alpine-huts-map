Alpine Hut Availability Map

Find mountain huts with recently checked free beds across the Alps. The map helps you quickly spot huts that can be booked via hut-reservation.org, instead of opening dozens of calendars by hand.

How to use it

Choose your arrival and departure dates, set the minimum number of places, and click a hut to see the availability calendar. Green means more free beds, yellow means only a few places left, red means full.

This is a small private project. The data may be incomplete, contain errors, or become outdated. The official reservation system of each hut is always the source of truth. The data should be updated early in the morning. There will be actual changes throughout the day.

If you find a mistake or something does not work, please let me know or open an issue:
